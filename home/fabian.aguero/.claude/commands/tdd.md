Activa el agente `tdd-guide` para guiar la implementación con TDD.

Argumentos esperados: $ARGUMENTS (nombre de la entidad o feature a implementar)

1. Identifica la entidad/feature: $ARGUMENTS
2. Delega al agente `tdd-guide` con el contexto completo
3. El agente generará los test cases a implementar
4. Sigue el ciclo estricto: RED → GREEN → IMPROVE
5. Al finalizar, verifica que la cobertura >= 80%
6. Ejecuta `/review` para validar el código resultante
